New Methods Challenge: toFixed() and valueOf()

About the toFixed() Method:

The toFixed() method formats a number using fixed-point notation.

Definition: This method returns a string representation of the number with exactly the specified number of digits after the decimal point.

Key Characteristics:

  • Returns a string, not a number.
  • Rounds the number if necessary.
  • Pads zeros if needed
  • Parameter range: 0-20 (some browsers support up to 100)
  • Syntax: number.toFixed(digits)

Common Use Cases: Currency formatting, percentage display, scientific measurements.

About the valueOf() Method:

The valueOf() method returns the primitive value of a Number object.

Definition: This method returns the primitive value (number) of a Number object allowing for type comparison and arithmetic operations.

Key Characteristics:

  • Returns the primitive numeric value
  • Called automatically by JavaScript in most operations
  • Essential for understanding type coercion
  • Syntax: number.valueOf()
  • Works with Number objects and primitive numbers

Common Use Cases: Type conversion, explicit value retrieval, understanding object vs primitive behavior.

String Methods Challenge: toUpperCase() and search()

toUpperCase() Method Definition:

The toUpperCase() method returns a string where all characters are converted to uppercase letters.

Key characteristics:

  • Does not modify the original string (strings are immutable in JavaScript).
  • Returns a new string with all uppercase characters.
  • Numvbers and special characters remain unchanged.
  • Syntax: string.toUpperCase()

search() Method Definition:

The search() method searches for a match between a regular expression or substring and the calling string.

Key characteristics:

  • Reeturns the index position of the first character of the match.
  • Returns -1 if no match is found.
  • Search is case-sensitive by default.
  • Can use regular expressions for more complex searches.
  • Syntax: string.search(searchValue)

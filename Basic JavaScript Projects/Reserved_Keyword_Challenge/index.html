JavaScript Reserved Word Variable and Constructor Function

1. Assigning a Value to a Reserved Word

Note: In modern JavaScript, you cannot use reserved words as varaible names in strict mode.

// Attempting to use a reserved word as a variable name // This will cause a syntax error in most cases try { // Using eval to demonstrate the concept (not recommended in practice) eval("var Function = 'This is not allowed';""); document.getElementById('reservedResult').innerHTML = "Successfully assigned value to 'function' variable"; } catch (error) { document.getElementById('reservedResult').innerHTML = "Error: Cannot use reserved words as variable names. " + error.message; }

2. Object Constructor Function

Creating a constructor function to create person object:

// Boject constructor function for creating Person objects funtion Person(name, age, occupation) { this.name = name; this.age = age; this.occupation = occupation; // Method attached to the constructor this.greet = function() { return "Hello, my name is " + this.name + " and I am " + this.age + " years old."; }; this.getDetails = function() { return this.name + " is a " + this.age + " year old " + this.occupation + "."; }; } // Creating instances of Person using the constructor var person1 = new Person("John Anderson", 28, "Software Engineer"); var person2 = new Person("Joe Mendy", 35, "Product Manager"); var person3 = new Person("Angelina Jolie", 30, "Designer"); // Displaying the created objects console.log(person1); console.log(person2); console.log(person3); // Using methods from the objects console.log(person1.greet()); console.log(person2.getDetails());

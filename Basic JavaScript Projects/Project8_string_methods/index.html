String Methods: concat() and slice()

About the concat() Method:

The concat() method is used to join two or more strings.

This method does not change the existing strings but returns a new string.

Click the button above to see the result of string concatenation.

About the slice() Method:

The slice() method extracts a section of a string and returns it as a new string.

Syntax: string.slice(start, end) where start is inclusive and end is exclusive.

Click the button above to see the result of the slice() method.

About the toString() Method:

The toString() method converts a number to a string.

Syntax: number.toString() or number.toString(base) where base is optional (default is 10).

Click the button above to see the result of the toString() method.

About the toPrecision() Method:

The toPrecision() method returns a string representing the number object to the specified precision.

Syntax: number.toPrecision(precision) where precision is the number of significant digits.

Click the button above to see the result of the toPrecision() method.

How they works:

concat(): Join strings together to create a new string.

slice(): Extracts a portion of a string based on index positions.

toString(): Converts to its string representation.

toPrecision(): Returns a string representation of number with specific precision.
